What is it?.

This plugin for GLPI is a subproject of Flyve MDM.

It provides a stand-alone web based administration and an interface integrated in GLPI. The later one may be the preferred by those who use GLPI for asset management and ticketing.


Features

Mobile management

Flyve MDM provides a range of policies to manage the mobile devices. Access to them whether through the Web Admin Dashboard or the GLPI interface:

  • Activate remotely encryption.
  • Real-time geolocation.
  • BluetoothÂ® and Wi-Fi connectivity control.
  • Control camera both front and facing.
  • Install and uninstall applications remotely.
  • Support to Bring Your Own Devices (BYOD).

Easy and simple

Flyve MDM counts with an intuitive and smooth interface which integrated with GLPI, facilitates the usage, specially for users who implement this IT and asset management software.

Security

Control and secure the mobile devices with the security policies. In the event of lost or theft protect private and sensitive information.

  • Remotely erase all content and settings on your device.
  • Remotely lock terminals.
  • Set the level of complexity of your passwords.
